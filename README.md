# Movies-Recommendation-API
* Flask app which recommends movie of similar kind as searched one
* Dataset used : https://www.kaggle.com/datasets/tmdb/tmdb-movie-metadata
* search for a movie and in response of which it will recommend 5 similar kind of movies
* running command : python3 app.py
* in Postman : Get : http://localhost:5000/recommend?movie=Superman
